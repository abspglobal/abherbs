Straight forward scenario:

  1. Choose color of the flower by clicking on image.
  2. Choose habitat by clicking on image.
  3. Choose number of petals by clicking on image.
  4. Choose flower from the list.
  5. Read about flower, check other images.
  6. Go back, choose another one until you'll find right one.
  7. Go back to the filter screen and choose different color, locality or number of petals.

Jumping around scenario:

You can jump between filter screens through left drawer where you can also see chosen values of attributes. When filter screen is loaded value of bounded attribute is deleted from the current filter. Counter in bottom right corner will show you how many flowers fulfil chosen criteria. Once you have set all three attributes scrollable list of flowers appears. If counter shows less than 21 flowers you can jump right to the list by pressing counter without setting remaining attributes. Long press on counter will clear the filter.

Available actions:

  1. Press on picture in filter screen will set value for correspondent attribute and jump to the next filter screen if attribute without value exists, otherwise it will jump to list of flowers which fulfill the filter.
  2. Press on image in the list of flowers will display flower's details.
  3. Press on thumbnail picture on the screen with details will display bigger picture bellow.
  4. Press on counter display list of flowers. It is allowed only when counter value is less than 21.
  5. Long press on counter will clear the filter.
  6. Swipe from left or press on "hamburger" button display left drawer.
  7. Press on one of filter options in drawer will display correspondent filter screen. Value of bounded attribute is deleted from the filter.
  8. Press on Preferences option in drawer will display Preferences.
  9. Press on Feedback option in drawer will offer opportunity to correct typos, mistakes, submit new translation or any other feedback.
  10. Press on Donation option will display Donation screen. Any amount of BTC or LTC is welcome.
  11. Press on Help option in drawer will display Help (this screen).
  12. Press on About option in drawer will display About application screen.

Preferences:

  1. Preferred language: Choose preferred language.
  2. Change localization (to preferred language): Change localization to preferred language.

Application takes localization from the device on which it's running. Unsupported localization display texts from default localization (English). Current version supports English, Slovak and Czech localizations. For all other languages default English localization will be used.

Information about flowers which are displayed on detail screen is different issue. Texts are downloaded from a server which currently supports only two languages: English and Slovak. If localization's language doesn't match preferred language Google Translate icon will appear on detailed screen just next to information about flower. Touch this icon to translate information to your localization's language using Google Translate. Touch again to return back information in preferred language.

Example 1: Jan is using Czech language (localization) on his phone. Application supports this localization so all application's text will be shown in Czech. Jan chooses Slovak as his preferred language in Settings (the application doesn't support Czech as preferred language yet). Application server will send information texts in Slovak because Jan didn't change application's localization (check box Change localization stays unchecked). Google Translate icon will appear just next to information on detail screen and Jan could translate texts from Slovak to Czech using Google Translate.

Example 2: Vlado is using English language (localization) on his phone. He chooses Slovak as his preferred language in Settings and he also change localization to Slovak for this application. Vlado will see application's texts and information texts in Slovak and no Google Translate icon will appear on detail screen.

Example 3: Jurij is using Russian language (localization) on his phone. Application doesn't support this localization so all application's text will be shown in English. Jurij chooses English as his preferred language in Settings (the application doesn't support Russian as preferred language yet). Application server will send information texts in English. Jurij didn't change application's localization (check box Change localization stays unchecked). Google Translate icon will appear just next to information on detail screen and Jurij could translate texts from English to Russian using Google Translate.
